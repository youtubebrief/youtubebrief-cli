# Beta access

Youtubebrief CLI/MCP is publicly installable from npm, but API-backed brief generation may require beta access or credits.

## Request access

https://youtubebrief.com/beta

## What to expect

- Public npm install is available.
- `yb --help`, `yb doctor`, dry-run batch planning, and local docs are available without spending credits.
- `yb brief`, paid batch execution, and MCP brief generation may require API access and remaining minutes.
- Credits may come from beta grants, prepaid minute packs, or operator-assisted rollout depending on availability.
- Live payment availability may vary while beta payment setup is finalized.

## Safe public wording

> Public npm install is available. Youtubebrief is still beta. Credits/API access is handled through beta access, prepaid minute packs, or operator-assisted rollout depending on availability.

Do not claim a full self-serve paid launch until live payment setup is explicitly confirmed.
